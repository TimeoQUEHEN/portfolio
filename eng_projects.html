Flatcraft

Flatcraft is a school project made in 2nd year of Lens' three year diploma in IT.
It consists of a mix between Minecraft and Terraria, and we, as the students, were to implement multiple functions, and overall, add content to the game, as we were given just the skeleton of the game's code.

    For instance
  • Generate the terrain
  • Mine the different cells in the map
  • Combine different resources
  • Create different creatures to inhabit the map.

Language : Java and its framework JavaFX

ParodieJO

Parodie JO is a school project made in 2nd year of Lens' three year diploma in IT.
The goal was to familiarize ourselves with Laravel and its capabilities, as well as Eloquent and Fortify, so as to learn about athentification and handling user's capabilities.

web-wise : routes, controllers, cookies and templates were used.

database-wise : Migrations, Models, Factories, Seeders and pivots were implemented.

Language : PHP and its framework Laravel

Superb PokePedia

Superb Pokepedia is a school project made in 2nd year of Lens three year diploma in IT.
The goal was to teach us to use an already existing API, and gather data using requests. To allow that, we were tasked with creating a website in which a user could enter the name of a pokemon, and the website would show all the relevant information about said pokemon. Adiitionnal tasks were to handle authentication, allow a user to acces their search history and favorite pokemons to easily acces them.

As of today, this project is not completed.

Language : Python and its framework Flask

Vaultify

Vaultify is a school project made in 2nd year of Lens three year diploma in IT.
Paired to a group of 2nd year students in "Mutimedia and information jobs" at Lens, We had to, during 36h, create a website where users could submit stories in the "where you are the hero" genre. Additionnaly, users can comment under the different stories, change their avatar, favorite stories, edit their stories and mark them as done. We then had to create templates, request the database and modify it, handle the different permissions.

Langage : PHP and its framework Laravel

Raytracing

Raytracing is a school project made in 2nd year of Lens three year diploma in IT.
throughout a week, we, as students, were tasked to creat an application that generated images based on a file. we had to create points, triangles, planes and sphere in order to make it work.

To generated an image (called scene), the user feeds a text file to the application. Inside the file is indicated the position of each point, the intensity of each light source, the color and reflectability of each surface.

Each part of this project was created by the students.

Language : Java

Puissance 4

Puissance 4 is a school project made in 1st year of Lens three year diploma in IT.
From the translation of the board-game "Connect 4", the game is, in of itself, a virtual version of it. It is played with two players, inputting actions via the keyboard.
The game is played through text printed onto the computer's terminal.

Language : Python

Jeu du chameau

Le jeu du chameau is a school project made in 1st year of Lens three year diploma in IT.
The player is placed in the shoes of an adventurer lost in the dessrt, who stole a camel to regain civilization. The player must then travel through 300km of desert while looking after their (and its camel's) health, without letting the group of nomads catch up to them.
The game is played through text printed onto the computer's terminal.

Language : Python
